## IPC Lab — TCP клиент/сервер
Проект лабораторной работы по межпроцессному взаимодействию: реализация простого обмена сообщениями через TCP (модель ping → pong).

## Структура проекта
src/
server/ — серверная часть
client/ — клиентская часть
include/ — заголовочные файлы 

## Git-процесс команды
- Никто не работает в main.
- Каждый участник работает в своей ветке:
Максим — feat/max/server 
Катя — feat/kat/client 
Маша — feat/mash/test 
- Все изменения отправляются через Pull Request.
- Слияние делается только в main после проверки.
